WSO2 HTTP Analytics Server development team is pleased to announce the release of WSO2 HTTP Analytics Server 1.0.0-m1. WSO2 HTTP Analytics Server 1.0.0-m1 is the single solution for HTTP traffic analysis for WSO2 Servers. WSO2 HTTP Analytics Server is powered by WSO2 Carbon, the SOA middleware component platform.

An open source product, WSO2 Carbon is available under the Apache Software License (v2.0)

You can download this distribution from http://wso2.com/platform and give it a try.

How to Run

  1. Extract the downloaded zip
  2. Go to the bin directory in the extracted folder
  3. Run the wso2server.sh or wso2server.bat as appropriate
  4. To access the management console, point your browser to the URL https://localhost:9443/carbon
  5. To access the HTTP Analytics dashboard, point your browser to the URL https://localhost:9443/portal/dashboards/http-analytics/
  6. Use "admin", "admin" as the username and password to login as an admin
  7. If you need to start the OSGi console with the server use the property -DosgiConsole when starting the server. The INSTALL.txt file found on the installation directory will give you a comprehensive set of options and properties that can be passed into the startup script

Key Feature of WSO2 HTTP Analytics Server

  • HTTP Traffic Analytics dashboard. This dashboard is capable of monitoring the following aspects of HTTP Analytics,
    • Number of requests, average response time, number of sessions and error count statistics
    • Browser, operating system and device type statistics
    • HTTP Response code statistics
    • Geolocation and language statistics
    • Context statistics
    • Referrer statistics
